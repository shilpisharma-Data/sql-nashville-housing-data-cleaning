# **Nashville Housing Data Cleaning Project (SQL)**--

# **Project Overview**--

 •  **This project focuses on cleaning and transforming the Nashville Housing dataset using MySQL**.




 • **The goal of this project was to convert the raw data into a clean, structured, and analysis-ready format using SQL queries**.













# **Dataset Information**--

•	**Dataset Name**: Nashville Housing Data

•	**Source**: Kaggle

•	**Format**: CSV

•	**Tool Used**:  MySQL Workbench 8.0




# **Tools & Technologies Used** --

•	**MySQL**

•	**MySQL Workbench 8.0**

•	**Git & GitHub**


# **Data Cleaning Steps Performed**

**The following operations were performed in this project**:

•	**Create Table (Raw Data)**

•	**Standardize Date Format**

•	**Fill Missing PropertyAddress (Self Join)**

•	**Split Property Address into  individual columns (Address, City) [USING SUBSTRING()]**

•	**Split Owner Address into individual columns (Address, City)**

•	**Standardize SoldAsVacant (Y/N → Yes/No)**

•	**Remove Duplicates**
